#### A. 优化算法 (Optimization Algorithms)
*解决问题的核心，特别是当问题复杂难以精确求解时。*

##### A1. 元启发式/智能优化算法 (Metaheuristics)
*   `遗传算法 (Genetic Algorithm - GA)`: 模拟生物进化过程，通过选择、交叉、变异全局搜索最优解。
*   `粒子群算法 (Particle Swarm Optimization - PSO)`: 模拟鸟群觅食行为，通过个体与群体经验在解空间中寻找最优解。
*   `模拟退火算法 (Simulated Annealing - SA)`: 模拟物理退火过程，通过概率性接受劣解来跳出局部最优。
*   `蚁群算法 (Ant Colony Optimization - ACO)`: 模拟蚂蚁寻找食物路径，通过信息素的正反馈机制求解最短路径问题。
*   `禁忌搜索算法 (Tabu Search - TS)`: 通过“禁忌表”记录已搜索过的解来避免重复，引导搜索到新的区域。
*   `NSGA-II (非支配排序遗传算法)`: 解决多个目标相互冲突（如成本与时间）的多目标优化问题的经典算法。
*   `MOPSO (多目标粒子群算法)`: 用于求解多目标优化问题的粒子群算法版本。

##### A2. 经典运筹学 (Classical Operations Research)
*   `线性规划 (Linear Programming - LP)`: 在线性约束下最大化或最小化线性目标函数。
*   `非线性规划 (Nonlinear Programming - NLP)`: 目标函数或约束条件中包含非线性函数的优化问题。
*   `整数规划 (Integer Programming - IP)`: 决策变量被限制为整数的线性或非线性规划。
*   `二次规划 (Quadratic Programming - QP)`: 目标函数为二次函数且约束为线性的优化问题。
*   `动态规划 (Dynamic Programming - DP)`: 将复杂问题分解为一系列简单的子问题来求解。
*   `分支定界法 (Branch and Bound)`: 用于求解整数规划和组合优化问题的精确算法。
*   `列生成算法 (Column Generation)`: 求解具有海量变量的超大规模线性规划问题的有效方法。
*   `车辆路径问题 (VRP) 启发式算法`:
    *   `节约算法 (Savings Algorithm/C-W算法)`: 求解VRP问题的经典启发式算法，通过合并路径来节约成本。
    *   `扫描算法 (Sweep Algorithm)`: 通过对节点进行角度排序来构造初始路径的启发式算法。
*   `指派问题 (Assignment Problem)`: (您已掌握`匈牙利算法`，属于此类)。

#### B. 预测与数据分析模型
*用于对问题中的未知参数进行预测，或从数据中提取规律。*

##### B1. 机器学习 (Machine Learning)
*   `随机森林 (Random Forest)`: 由多棵决策树构成的集成学习模型，用于分类和回归。
*   `梯度提升决策树 (GBDT)`: 通过迭代训练一系列弱学习器（决策树）来构建的强大预测模型，包括 `XGBoost` 和 `LightGBM`。
*   `支持向量机 (Support Vector Machine - SVM)`: 在特征空间上寻找最优超平面进行分类或回归的监督学习模型。
*   `逻辑回归 (Logistic Regression)`: 广义线性模型，常用于解决二分类问题。
*   `决策树 (Decision Tree)`: 基于树状结构进行决策的分类和回归模型。
*   `K-均值聚类 (K-Means Clustering)`: 将数据点划分为K个不同簇的无监督聚类算法。
*   `层次聚类 (Hierarchical Clustering)`: 通过构建簇的层次结构来进行聚类的算法。
*   `DBSCAN`: 一种基于密度的聚类算法，能有效处理噪声并发现任意形状的簇。
*   `朴素贝叶斯 (Naive Bayes)`: 基于贝叶斯定理和特征条件独立假设的简单高效分类器。

##### B2. 神经网络与深度学习 (Neural Networks & Deep Learning)
*   `BP神经网络 (Backpropagation NN)`: 使用反向传播算法进行训练的多层前馈神经网络。
*   `循环神经网络 (RNN)`: 专门用于处理序列数据（如时间序列）的神经网络。
*   `长短期记忆网络 (LSTM)`: RNN的一种变体，有效解决了长序列依赖问题。
*   `图神经网络 (GNN)`: 直接在图结构数据上进行学习和推理的神经网络模型。
*   `卷积神经网络 (CNN)`: 主要用于图像识别，但在特定序列数据中也可用于特征提取。

##### B3. 经典统计与时序模型
*   `ARIMA时间序列模型`: 用于分析和预测时间序列数据的经典统计模型。
*   `灰色预测模型 (Grey Prediction - GM(1,1))`: 在信息不完全的情况下，对数据进行短期预测的有效模型。
*   `指数平滑法 (Exponential Smoothing)`: 一种加权平均的时间序列预测方法。
*   `Prophet模型`: Facebook开源的强大且易用的时间序列预测工具。
*   `马尔可夫预测模型 (Markov Prediction)`: 基于马尔可夫链的状态转移概率进行未来状态预测。
*   `最小二乘法拟合 (Least Squares Fitting)`: 通过最小化误差平方和来寻找数据最佳函数匹配的拟合方法。

#### C. 综合评价与决策模型
*用于多准则、多方案的排序与选择。*

*   `层次分析法 (AHP)`: 将复杂决策问题分解为层次结构，通过两两比较确定权重。
*   `熵权法 (Entropy Weight Method)`: 利用信息熵客观地确定各指标权重的评价方法。
*   `TOPSIS综合评价模型`: 通过计算各方案与最优解和最劣解的距离来进行排序。
*   `基于熵权法对Topsis模型的修正`: 结合熵权法的客观赋权与TOPSIS的排序能力进行综合评价。
*   `模糊综合评价`: 应用模糊数学理论，对受多种因素影响的事物进行综合评判。
*   `网络分析法 (ANP)`: AHP的扩展，适用于指标间存在相互依赖关系的网络结构。
*   `VIKOR方法`: 一种基于折衷解的多准则决策方法，侧重于最大化群体效用和最小化个体遗憾。

#### D. 仿真、随机与系统模型
*用于模拟随机过程、动态演化和复杂系统。*

*   `蒙特卡洛模拟 (Monte Carlo Simulation)`: 通过大量随机抽样来解决计算问题和模拟随机过程。
*   `排队论 (Queuing Theory)`: 对服务系统中排队等待现象进行数学分析的理论。
*   `元胞自动机 (Cellular Automaton)`: 在离散空间和时间上模拟复杂系统（如交通流、火灾蔓延）演化的模型。
*   `Agent-based建模 (多智能体仿真)`: 通过定义微观个体（Agent）的行为规则来模拟宏观系统现象。
*   `传染病模型 (SIR, SEIR等)`: 描述传染病在人群中传播动态的数学模型。
*   `人口增长模型 (Logistic Model)`: 描述在资源有限环境下种群数量增长规律的模型。

#### E. 数据处理与分析
*建模前对数据进行预处理或降维。*

*   `主成分分析 (PCA)`: 将高维数据转换为低维数据的线性降维技术。
*   `插值算法 (Interpolation)`: 在离散数据点之间构建新数据点的数学方法。
*   `小波分析 (Wavelet Analysis)`: 在时域和频域上同时分析信号的强大工具。

#### F. 特定应用领域
*   `CT图像重建算法`: 从投影数据重建断层图像的系列算法（如滤波反投影法）。
